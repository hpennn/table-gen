# 表格生成器 (Table Generator)

> 免费在线表格生成器 - 快速创建、编辑、导入导出表格数据

🔗 在线体验：[https://table-gen.pages.dev](https://table-gen.pages.dev)

## 功能特性

- 📝 **在线编辑** - 可视化表格编辑器，支持增删行列、直接编辑单元格
- 📋 **文本转表格** - 粘贴 CSV/Tab 分隔文本，自动生成表格
- 📂 **导入文件** - 支持 CSV 和 Excel (.xlsx) 文件导入
- 📥 **导出功能** - 支持导出为 CSV 和 Excel (.xlsx) 格式
- 🔒 **隐私安全** - 纯前端处理，数据不上传服务器
- 📱 **响应式设计** - 适配桌面和移动端

## 技术栈

- **HTML5** + **Tailwind CSS** (CDN)
- **SheetJS (xlsx)** - Excel/CSV 读写
- **Cloudflare Pages** - 静态托管

## 本地运行

直接用浏览器打开 `index.html` 即可，无需任何服务器。
